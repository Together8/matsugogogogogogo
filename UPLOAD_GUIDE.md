# GitHub Pages 上傳教學

## 最穩做法

請建立一個全新的 Repository，不要沿用舊專案，避免舊快取或舊檔案影響。

## 需要上傳的檔案

只需要兩個：

1. index.html
2. README.md

## 注意

請確認 `index.html` 是放在 Repository 最外層，不要放在資料夾裡。

正確：

matsugogogo/
- index.html
- README.md

錯誤：

matsugogogo/
- matsu_trip/
  - index.html

## 如果上傳後還是舊畫面

請使用：
- 無痕視窗
- Ctrl + F5 強制重新整理
- 手機瀏覽器清除快取
